# OOTD JANG

### Introduction
OOTD JANG is clothing management program with OOTD(Outfit Of The Day).

### Function
1. Add new cloth
2. Delete cloth
3. Show closet
4. Add OOTD
5. Show OOTD
6. Make txt file of OOTD
7. Exit

### How to install
```
$ git clone https://github.com/think-jin/pg_project.git

$ cd pg_project/

$ make

$ ./ootd
```
